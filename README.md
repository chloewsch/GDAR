# GDAR

# Code associated with: Variability, drivers, and utility of genetic diversity-area relationships in terrestrial vertebrates 
## Schmidt et al. 2025 EcoEvoRxiv (doi.org/10.32942/X2Q34P)

File descriptions: <br>
- <b>functions_git.R:</b> functions needed to run analyses in other scripts. <br>
- <b>GDAR_script_git.R:</b> generate artificial area samples to construct and plot GDARs and estimate scaling exponents (z). <br>
- <b>interaction_models_git.R:</b> fit and plot models relating genetic diversity to area, FST, and their interaction. <br>
- <b>trait_data_git.R:</b> compile species trait data. <br>
- <b>training_test_git.R:</b> test the predictability of FST and scaling exponents z using species traits. <br>
